# Double Pendulum Simulation

A Python-based simulation of a double pendulum system demonstrating nonlinear dynamics and chaotic motion.

## Overview

The double pendulum is a classic example of a deterministic system that exhibits chaotic behavior. Small changes in initial conditions can lead to drastically different trajectories.

This project numerically solves the equations of motion and visualizes the resulting pendulum motion.

## Features

- Real-time animation of the double pendulum
- Visualization of pendulum trajectories
- Adjustable physical parameters
- Demonstration of chaotic behavior
- Physics-based numerical simulation

## Physics Concepts

- Classical Mechanics
- Lagrangian Dynamics
- Nonlinear Systems
- Chaotic Motion
- Conservation of Energy

## Technologies Used

- Python
- NumPy
- Matplotlib

## Numerical Concepts

- Euler numerical Method
